Ключевые изменения в simplex_method:

* Полное приведение к каноническому виду Ax = b, x ≥ 0: * 

<= → добавляем положительную **sлаг-**переменную

>= → умножаем на -1 + искусственная переменная

== → искусственная переменная

Фазовый метод подготовлен:

c_phase1 минимизирует сумму искусственных

Начальный базис: искусственные переменные

Вывод диагностики показывает структуру канонической формы

* Ключевые изменения - полная реализация вспомогательной задачи:

✅ Вспомогательная целевая функция: min w = ∑aᵢ где aᵢ - искусственные переменные

✅ Матрица ограничений: [A | I] x_aug = b с правильным добавлением:

≤ → слэковые переменные ✅

≥ → -A + искусственная ✅

= → искусственная ✅

✅ Начальное базисное решение: искусственные aᵢ = bᵢ ≥ 0 ✅

✅ Диагностический вывод показывает структуру фазы 1

* Добавлен полный симплекс-алгоритм для фазы 1:

Симплекс-таблица с базисом и z-строкой

Правило входа: максимальный отрицательный коэффициент в z-строке

Правило выхода: минимальное положительное отношение b/a

Pivot операция с нормализацией

Диагностический вывод каждой итерации

* Добавлен ПОЛНЫЙ ПЕРЕХОД К ФАЗЕ 2:

transition_to_phase2() - проверяет отсутствие искусственных в базисе + заменяет целевую функцию

solve_phase2() - решает оригинальную задачу с Bland's rule (без циклов)

extract_original_solution() - возвращает только исходные переменные x1,x2,...
